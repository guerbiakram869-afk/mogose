# Mangoose Demo

Simple Mongoose demo showcasing common CRUD operations on a `Person` model using MongoDB Atlas.

Maintainer: Akram

## Prerequisites
- Node.js 18+
- A MongoDB Atlas connection string

## Setup
1. Copy `.env.example` to `.env` and set `MONGO_URI`.
2. Install dependencies:
	- `npm install`

## Run
- `npm start`
